# Kiñit - Ethiopian Scale MIDI Transformation

A Max for Live MIDI transformation tool for Ableton Live 12 that snaps incoming MIDI notes to Ethiopian music scales.

## Installation

1. **Locate your Max for Live User Library**
   - Windows: `Documents\Ableton\Live\User Library\MIDI Tools\Max Transformations`
   - Mac: `~/Documents/Ableton/Live/User Library/MIDI Tools/Max Transformationse`

2. **Copy the files**
   - Copy `Kiñit Scale` folder to your Max for Live User Library

3. **Refresh Live's browser**
   - Close and reopen Ableton Live, or right-click in the browser and select "Refresh"

## Usage

1. **Add to a MIDI clip**
   - Open a MIDI clip in the Clip View
   - Click the Transform or Generate panel in the bottom of the clip view
   - Find "Kiñit" in the User folder and select it

2. **Select a scale**
   - Use the Scale dropdown to choose from:
     - Tizita Major
     - Tizita Minor
     - Bati Major
     - Bati Minor
     - Ambassel Major
     - Ambassel Minor
     - Anchihoye

3. **Set the root note**
   - Use the Root parameter to set the root note (C = 0, C# = 1, etc.)

4. **Apply transformation**
   - Click "Apply" in the clip view to transform the notes
   - Or enable "Auto" to transform in real-time as you play

## Features

- Snaps incoming notes to the nearest note in the selected Ethiopian scale
- Supports 7 traditional Ethiopian scales (pentatonic and hexatonic)
- Configurable root note
- Visual feedback in the clip view

## About the Scales

Ethiopian music uses unique modal systems distinct from Western scales:

- **Tizita**: The most common scale, similar to a major pentatonic with an added note
- **Bati**: Traditional scale used in Bati music
- **Ambassel**: Minor-key scale common in Ethiopian jazz
- **Anchihoye**: Unique scale with a distinctive interval pattern
